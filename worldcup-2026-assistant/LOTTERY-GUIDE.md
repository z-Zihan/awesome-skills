# 体彩选购指南

> 本文件从 worldcup-2026-assistant SKILL.md 拆分而来。当用户询问体彩相关问题时，读取本文件获取详细指南。

## 功能四：体彩选购指南

### 触发场景

- "今天的体彩"
- "能买什么比赛"
- "体彩赔率"
- "单关串关推荐"
- "体彩购买时间"

### 体彩基础知识

- **体彩** = 中国体育彩票（唯一合法足球竞猜渠道）
- **竞彩足球** = 体彩中的足球竞猜产品，共5种玩法
- **胜平负** = 猜主队胜/平/负，最基础玩法
- **让球胜平负** = 加上让球数的胜平负
- **比分** = 猜具体比分（仅90分钟+补时），31种选项，赔率最高
- **总进球** = 猜比赛总进球数（0/1/2/3/4/5/6/7+球，共8个选项）
- **半全场胜平负** = 猜上半场结果+全场结果的组合（9种：胜胜/胜平/胜负/平胜/平平/平负/负胜/负平/负负）
- **单关** = 单独买1场比赛
- **串关/过关** = 猜多场比赛，全对才赢，赔率累乘
- **混合过关** = 一注里面混不同玩法串起来

### 半全场详解

猜上半场结果 + 全场结果，共9种组合：

| 简称 | 上半场 | 全场 | 典型场景 |
|------|--------|------|---------|
| **胜胜** | 主队领先 | 主队赢 | 强队从头压到尾 |
| **胜平** | 主队领先 | 平局 | 主队先进球后被逼平 |
| **胜负** | 主队领先 | 客队赢 | 大逆转，赔率最高 |
| **平胜** | 平局 | 主队赢 | 下半场发力 |
| **平平** | 平局 | 平局 | 全场闷平 |
| **平负** | 平局 | 客队赢 | 僵持后客队破门 |
| **负胜** | 客队领先 | 主队赢 | 逆转 |
| **负平** | 客队领先 | 平局 | 主队追平 |
| **负负** | 客队领先 | 客队赢 | 客队从头赢到尾 |

- 强队打弱队最常见的两个结果：**胜胜**（全程碾压）、**平胜**（上半场僵持下半场破局）
- **支持单场投注**，赔率比胜平负高
- ⚠️ 跟比分/总进球一样只算90分钟+补时

### 混合过关详解

把不同玩法串在一注里。例如：
> 荷兰「胜平负-胜」+ 德国「半全场-胜胜」+ 厄瓜多尔「让球-让胜」

| 规则 |
|------|
| 最多 8串1（含比分/半全场时降至 4串1） |
| 同一场比赛的**不同玩法可以混串** |
| 同一场比赛的**互斥结果不能串**（如同场「胜」和「平」） |
| 不同比赛的任意玩法可自由组合 |

- ⚠️ 不做主动推荐，除非用户明确问

### ⚠️ 玩法规则速查

| 问题 | 答案 |
|------|------|
| 几块钱一注？ | **2元**，最低投注单位 |
| 最多串几场？ | 胜平负/让球 **8串1**；比分/半全场 **4串1**；总进球 **6串1** |
| 混合过关含比分/半全场？ | 最多 **4串1** |
| 比分/半全场能单关吗？ | ✅ 天然支持单场，不用 dg |
| 胜平负/让球能单关吗？ | 只有 dg=1 的场次才能单关，dg=0 必须串 |
| 同一场能同时买比分和让球吗？ | ❌ 每场一注只能选一种玩法 |
| 同一场「胜」和「平」能串吗？ | ❌ 互斥结果不能串 |
| 4串1双选（每场2个）多少注？ | 2⁴ = **16注 × 2元 = 32元** |
| 8串1单挑每场1个多少注？ | 1⁸ = **1注 × 2元 = 2元** |
| 8串247是什么意思？ | 8场全组合容错：2串1~8串1共247注，**494元** |
| 容错过关是什么意思？ | 如 8串9 = 允许错1场（买8个7串1），比8串1贵但容错 |
| 所有玩法算加时赛吗？ | ❌ **只算90分钟+伤停补时**，加时/点球不计 |
| 强弱悬殊场怎么买？ | 胜平负可能不开售，只能买让球或比分/半全场 |
| 赔率哪里查最准？ | 官方 sporttery.cn → zgzcw → autoglm-websearch |

**⏰ 关键规则**
所有竞彩足球玩法（胜平负、让球、比分、总进球、半全场）一律**只算90分钟+伤停补时**，淘汰赛也一样！加时赛和点球均不计入。如需猜冠军/晋级方，需专门的"冠军竞猜"玩法

### 单关与串关详解

**什么是单关**
单关 = 只猜1场比赛，买了就走。简单直接，风险可控

**什么是串关**
串关 = 同时猜2场以上，全对才赢，赔率相乘。如2串1 = 猜2场全对，赔率=场1赔率×场2赔率

**⚠️ 单关规则（重要）**
**比分、总进球、半全场**：天然支持单场投注，可以单独买1场
**胜平负、让球胜平负**：默认至少2串1，只有体彩中心指定的"单关场次"才能单独买

体彩中心决定哪些场开胜平负单关，不是自动全开
- **强弱悬殊场通常不开单关**：赔率太低（如1.10-1.22），买100才赚十几块
- **豪门焦点战容易开单关**：关注度高、竞猜热度大
- 有些场连胜平负玩法都不开售，只能买让球玩法

**串关上限**
- 胜平负 / 让球胜平负：最多 8串1
- 总进球：最多 6串1
- 比分 / 半全场：传统过关 2串1、3串1；自由过关最多 4串1
- 混合过关：最多 8串1（含比分/半全场时降至4串1）

### 🎯 比分4串1双选战术

> 实战验证的高赔率比分玩法策略。投入 16注×2元=32元，全中回报可达百倍以上。

**核心思路：**
1. 每场选 **2 个比分**，覆盖最可能的两个结果
2. 4 串 1 = 2×2×2×2 = **16 注**，每注 2 元 = 总投入 32 元
3. 比分赔率 5-12 倍，4 场相乘 → 数千倍赔率

**选比分原则：**
- **强队零封**：如德国 vs 弱旅 → 选 2:0 + 3:0
- **实力接近**：两边各押一个小比分 → 1:0 + 0:1（不猜输赢，两边覆盖）
- **可能平局**：1:1 + 0:0 或 1:1 + 0:1
- **原则**：2 个选项必须覆盖不同的比赛走向，不能两个都是主队赢

**实战案例（2026-06-20 实际中奖票）：**

| 场次 | 比分选项 | 策略 |
|------|---------|------|
| 美国 vs 澳大利亚 | 2:0 @7.00 + 2:1 @5.50 | 美国赢2球 |
| 苏格兰 vs 摩洛哥 | 1:1 @5.85 + 0:1 @6.05 | 平 or 摩洛哥小胜 |
| 巴西 vs 海地 | 2:0 @7.20 + 3:0 @5.50 | 巴西零封大胜 |
| 土耳其 vs 巴拉圭 | 1:0 @6.60 + 0:1 @11.50 | 两边覆盖，无论谁1-0都中 |

**结果**：2:0 × 0:1 × 3:0 × 0:1 全中 → 7.00×6.05×5.50×11.50 = **2678 倍** → 32元变 **5357 元**

**⚠️ 比分投注死亡判断规则（2026-06-28 踩坑）**：
- 比分投注一旦场上总进球数超过所选比分的总进球数，该注已死！
- 如买1:1（总2球），场上2-1（总3球）→ 已死，不可能回到1:1
- 判断步骤：
  1. 当前总进球 > 所选比分总进球 → **已死**
  2. 当前总进球 = 所选比分总进球但比分不匹配 → **已死**
  3. 当前总进球 < 所选比分总进球 → **还活着**
- ❌ 2026-06-28 踩坑：用户买克罗地亚vs加纳1:1，场上2-1时说'还有机会追平'，实际已经死了

**何时推荐：**
- ✅ 当天有 4 场以上比赛可买
- ✅ 多场强弱对决，强队零封概率高
- ✅ 有 1-2 场实力接近的比赛，需双选覆盖
- ❌ 强弱过分悬殊（如德国 vs 库拉索，胜平负不开售，比分也可能不开）。**输出前必须核实该场赔率是否开售**
- ❌ 当天比赛不足 4 场
- ❌ 用户只想少量投入（32元门槛）

**输出模板：**

```
🎯 比分4串1双选 · 32元
 ・🇦🇷 阿根廷 vs 奥地利：2:0 / 2:1
 ・🇫🇷 法国 vs 伊拉克：3:0 / 4:0
 ・🇳🇴 挪威 vs 塞内加尔：2:1 / 2:0
 ・🇯🇴 约旦 vs 阿尔及利亚：0:2 / 1:2
```

**格式规则：**
- 标题行固定：`🎯 比分4串1双选 · 32元`
- 每行一场：` ・🇺🇸 主队 vs 🇨🇳 客队：比分1 / 比分2`
- 主队国旗 + 主队 vs 客队国旗 + 客队 + 冒号 + 两个比分用 `/` 分隔
- 不再使用表格、赔率标注、策略说明（简洁为主）
- 可在方案后附上理论奖金计算

### 🎯 比分2串1四选战术

> 4串1双选的「稳」字版本。同样投入32元，只需2场全中，命中率高得多。

**核心思路：**
1. 只选 **2 场**比赛，每场选 **4 个比分**
2. 2串1 = 4×4 = **16注**，每注2元 = 总投入32元（跟4串1双选一样的钱）
3. 只需2场全中就赢，比4场全中现实得多
4. 每场4个比分覆盖面广，淘汰赛低进球场景命中率极高

**4串1双选 vs 2串1四选对比：**

| 维度 | 4串1双选（32元） | 2串1四选（32元） |
|------|------------------|------------------|
| 场次数 | 4场 | 2场 |
| 每场选项 | 2个比分 | 4个比分 |
| 总注数 | 16注 | 16注 |
| 全中条件 | 4场全中 | 2场全中 |
| 命中概率 | 低（4场连中） | 高（2场连中+每场4选覆盖广） |
| 赔率倍数 | 数千倍 | 数十倍 |
| 理论奖金 | 几千~上万 | 几十~两三百 |
| 适合场景 | 娱乐搏大 | 稳健增收 |

**选比分原则（淘汰赛优化）：**
- 淘汰赛比分集中在低进球：0:0、0:1、0:2、1:0、1:1、2:0、2:1
- 每场选4个 = 覆盖70%+可能比分
- **选场标准**：选比分范围窄、好锁定的比赛
  - ✅ 强弱分明但弱队防守还行（比分大概率0:0~0:2）
  - ✅ 实力接近但都偏保守（比分大概率1:0~1:1）
  - ❌ 比分跨度大的场次（如德国vs弱旅可能2:0/3:0/4:0，4个选项兜不住）
  - ❌ 双方都对攻型（比分不可控）

**选场逻辑：**
- 优先选**淘汰赛低进球特征明显**的比赛（弱队死守、强队不冒险）
- 优先选**比分范围窄**的比赛（2-3个比分就能覆盖80%概率）
- 避开**比分跨度大**的比赛（强队可能大胜也可能小胜，4选兜不住）

**输出模板：**

```
🎯 比分2串1四选 · 32元
 ・🇿🇦 南非 vs 🇨🇦 加拿大：0:0 / 0:1 / 0:2 / 1:1
 ・🇧🇷 巴西 vs 🇯🇵 日本：1:0 / 2:0 / 2:1 / 1:1
```

**格式规则：**
- 标题行固定：`🎯 比分2串1四选 · 32元`
- 每行一场：` ・🇺🇸 主队 vs 🇨🇳 客队：比分1 / 比分2 / 比分3 / 比分4`
- 4个比分用 `/` 分隔
- 简洁为主，可在方案后附理论奖金估算

**何时推荐2串1四选 vs 4串1双选：**
- ✅ 推荐2串1四选：用户想稳一点、淘汰赛低进球场景、比分范围窄的比赛
- ✅ 推荐4串1双选：用户想搏大的、有多场强弱分明的好场次
- ✅ 可以两个方案都给，让用户自己选
- ❌ 当天不足2场比赛

**单关查询方式**
1. **sporttery.cn 官网截图（唯一权威来源）** ⚠️ 最优先
   - 让用户发 sporttery.cn 的截图，按截图判断
   - URL: `https://m.sporttery.cn/mjc/jsq/zqspf/`（需浏览器打开）
   - "单"标在[0]行→胜平负可单关；"单"标在让球行→让球可单关；无标记→只能串关
2. **autoglm-websearch** 搜索（次选，可能不准确）
   - 搜索词：`竞彩足球 X月X日 单关场次` 或 `竞彩足球 X月X日 赔率`
3. **❌ 500.com 和 zgzcw 的单关数据不可靠**，禁止用于判断单关
4. **中国体彩APP**（最权威，手机端）

**⚠️ 胜平负和让球胜平负是两个完全独立的玩法**：
- 胜平负可单关 ≠ 让球也能单关！必须分别确认
- 胜平负开售 ≠ 让球也开售！必须分别确认
- 每个玩法有独立的"单"标记，位置决定该玩法能否单关
- 推荐方案时必须校验：推荐的每个玩法是否真实可单关/可串关

**⚠️ 单关判断红线规则**：
- **禁止凭印象判断单关**：不能因为「强弱悬殊」「已出线轮换」就默认只能串关
- **禁止用 500.com/zgzcw 判断单关**：这两个来源的单关数据都与 sporttery 官网不一致过
- **必须逐场逐玩法确认**：每场分别确认胜平负是否开了、是否可单关、让球是否开了、是否可单关
- **历史单关记录见 `worldcup-data/lessons-learned.json` 的 `singleGateRecords`**

**让球玩法规则**：
- 让球 = 假设计分时给弱队加N球再判胜负
- **让球方减让球数，受让方加让球数**
- **让胜** = 加球后该队赢
- **让平** = 加球后平局
- **让负** = 加球后该队输
- ⚠️ **让胜 ≠ 输1球也中！** 输让球数恰好=让平，不是让胜！
- **详细举例见 `worldcup-data/lessons-learned.json` 的 `handicapLessons`**

**让球玩法说明**
让球 = 假设强队开场就落后N球，再判胜负
例如 法国-1：如果实际2-0 → 让球后1-0 → 法国让球胜；如果2-1 → 让球后1-1 → 让球平
赔率格式：让球胜 / 让球平 / 让球负
强弱差距越大，让球数越多（如伊拉克+2 vs 挪威）

### 📊 期望值分析

> 基于价值投注理论，用数学期望值过滤下注决策。核心思想：只有当 AI 估计的真实概率 × 赔率 > 1 时，长期才有正收益。

**核心公式：**

1️⃣ **虚拟胜率（庄家隐含概率）= 1 / 赔率**
   - 如阿根廷胜赔率 1.32 → 虚拟胜率 = 1/1.32 = 75.75%
   - 三项虚拟胜率之和 > 100%，超出部分为庄家抽水（通常 8-15%）

2️⃣ **期望值 = (真实概率 × 赔率) - 1**
   - 「真实概率」= 本 skill 的 AI 分析给出的胜率估计
   - 期望值 > 0 = 有长期价值，期望值 < 0 = 长期亏钱

**决策标准：**
- 期望 < 0 → ❌ **不买**
- 期望 0% ~ 5% → ⚠️ **观望**（价值不够）
- 期望 > 5% → 🟡 **轻仓**
- 期望 > 15% → 🟢 **正常投入**

**真实概率估计方法：**
- 根据预测分析的综合维度（实力差、近期状态、交锋记录、核心球员状态）给出百分比
- 高置信度预测 → 胜率 70%+；中高 → 55-70%；中 → 40-55%；中低 → 30-40%；低 → <30%
- 每个赔率选项都要给出对应概率，三项概率之和应接近 100%（不是赔率反推的 112%，而是去抽水后的真实估计）

**串关期望值：**
- 串关期望 = (真实概率1 × 真实概率2 × ... × 赔率1 × 赔率2 × ...) - 1
- ⚠️ 串关抽水是指数叠加的，2串1实际抽水约 24%，3串1约 36%，串越多抽水越重
- 因此串关的期望值门槛更高，建议 > 10% 才考虑

**让平高赔率检测规则：**
- 当让球盘中让平赔率 ≥ 3.40 时，触发检测
- 如果 AI 概率三选项接近均分（各约 30-36%），让平期望值天然为正（因为 0.33 × 3.40 = 1.122 > 1）
- 此时让平是一个价值投注，即使 AI 对胜负判断模糊，高赔率本身提供了价值
- ⚠️ 但风险在于：让平需要强队恰好赢让球数，一场没中就全废，串关风险极高
- 建议：让平高赔率可作为单关或 2 串 1 轻仓尝试，不建议多场串关

**使用场景：**
- ✅ 每场推荐都附带期望值计算，作为「买不买」的最终过滤层
- ✅ 串关方案也要算总期望值
- ✅ 用户自行决策时，建议参考期望值而非单纯看赔率高低
- ✅ 让平赔率 ≥ 3.40 时自动触发高赔率检测
- ❌ 不要唯期望值论——AI 概率估计本身有误差，期望值只是辅助工具

**示例：**
```
📊 期望值分析 · 阿根廷 vs 奥地利

不让球 [0]：
• 阿根廷胜：赔率 1.32 | AI概率 80% | 期望 = 0.80×1.32-1 = +5.6% 🟡轻仓
• 平局：赔率 4.17 | AI概率 15% | 期望 = 0.15×4.17-1 = -37.4% ❌
```

### 🔴 解析防错规则

**zgzcw.com 的 HTML 中，每场比赛行内包含多个玩法的赔率数据（胜平负、让球、比分、总进球、半全场），必须严格区分，禁止混用！**

**常见错误：**
- ❌ 把比分玩法（22.76 等高赔值）当作让球赔率 → 导致预测回报虚高
- ❌ 把半全场数值混入胜平负

**正确做法：**
1. 每场比赛的前 3 个数值（A/B/C）是**胜平负**（主胜/平/客胜）
2. 接着 3 个数值（D/E/F）是**让球胜平负**（让胜/让平/让负）
3. 后续数值是比分/半全场/总进球，只在用户明确问这些玩法时才使用
4. **数值校验**：让球赔率应合理（-2 让球胜通常 1.5~2.5，不可能出现 20+）。如发现异常高值，一定是取错了玩法列！
5. **用户出票后以票面为准**：如果用户发了实际彩票照片，票面上的赔率 > 一切抓取数据。**立即以票面数据覆盖账本！**

### 体彩查询要点

- **最权威来源**：`https://m.sporttery.cn/mjc/jsq/zqspf/`（中国体彩官方胜平负计算器），需用 autoglm-browser-agent 打开（JS 渲染），curl 不可用
- **备用来源**：`https://cp.zgzcw.com/lottery/jchtplayvsForJsp.action?lotteryId=47&type=jcmini`（足彩网），curl + 浏览器 UA 可抓 HTML，解析 dg/让球/赔率
- ⚠️ 体彩官方页面一次只显示 2-3 场，需滚动查看全部比赛

- **赔率**：实时查询，体彩赔率会变化，以购买时为准
- **单关信息**：并非所有比赛都开单关，必须查询确认。查询优先级：autoglm-websearch > zgzcw.com(web_fetch，如可用) > 中国体彩APP
- **胜平负未开售**：强弱悬殊场可能连胜平负都不开售，只能买让球玩法
- **销售时间**：工作日和周末不同，注意截止时间
- **让球数**：根据双方实力差距设定，注意让球后赔率变化
- **比分玩法**：31种比分选项（主队胜13种、平5种、主队负13种，含"胜其他""平其他""负其他"）。**支持单场投注**。赔率最高但命中率低，属于"以小博大"娱乐型，推荐时占比要小，点到为止
- **半全场胜平负**：9种组合（上半场胜平负 × 全场胜平负），赔率仅次于比分，同样支持单场投注
- **总进球**：8个选项（0-7+球），支持单场投注
- **⏰ 重要规则**：所有竞彩足球标准玩法（胜平负、让球、比分、总进球）一律**只算90分钟+伤停补时**，淘汰赛也一样！加时赛和点球均不计入。如需预测冠军/晋级方，需专门的"冠军竞猜"玩法，不是普通胜平负
- **赔率来源**：必须从中国体彩渠道获取（zgzcw.com等），不用国际盘口作为体彩赔率
- **只推荐中国体彩合法渠道**：不提及任何境外博彩平台

### ⚠️ 投注方案校验规则

**用户发来投注方案后，必须逐注校验每个玩法能否实际出票，禁止跳过校验直接说"方案没问题"！**

**校验步骤（必须全部执行，缺一不可）**：
0. **逐注拆解**：把用户方案的每一注拆开，记录：哪场比赛、哪个玩法（胜平负/让球/比分）、单关还是串关、金额
1. **逐场逐玩法确认单关标记**：
   - 从 sporttery 截图/用户确认/autoglm-websearch 获取每场比赛的单关标记
   - **胜平负和让球是两个独立玩法**，必须分别确认：
     - 胜平负[0]行有"单"标记 → 胜平负可单关
     - 让球行有"单"标记 → 让球可单关
     - 无"单"标记 → 该玩法只能串关
   - **❌ 禁止假设**：不能因为胜平负可单关就默认让球也能单关！
2. **检查每场比赛的玩法是否开售**：
   - 胜平负显示"未"或"--"=未开售，只能买让球
   - 让球显示"未"或"--"=未开售（罕见）
3. **逐注校验**：
   - 单关注：检查该场该玩法是否有"单"标记。无标记→**立即警告**「⚠️ XX这场XX玩法不能单关，只能串关！」
   - 串关注：检查每场该玩法是否开售。未开售→**立即警告**
   - 比分/总进球/半全场：天然支持单场，不受单关限制
4. **互斥检查**：同一场比赛的多注之间不能买互斥结果
5. **输出校验结果**：
   - ✅ 全部通过 → 方案可行
   - ❌ 有问题 → 逐条列出问题 + 给出修改建议

**以后流程**：用户发方案 → 逐注拆解 → 逐注校验单关/串关/开售状态 → 全部通过才说"方案可行" → 有问题立即警告并给修改建议

### 📡 实时比分查询

**用户问比赛结果时，不能只依赖 autoglm-websearch 搜索（搜索索引有延迟，刚结束的比赛可能搜不到）。必须直接抓取实时比分页面！**

**查询策略（按优先级）：**
1. **ESPN API（首选）**：`curl -s "https://site.api.espn.com/apis/site/v2/sports/soccer/fifa.world/scoreboard"`
   - 返回 JSON 格式，含 `name`（队名）、`score`（比分）、`status`（比赛状态/时钟）、`clock`（已过秒数）
   - **已验证可用**，数据准确无需怀疑
2. **autoglm-websearch**：搜索「X vs Y 比分 全场比赛结果」，但必须标注「搜索结果可能有延迟」
3. **81tiyu 比分直播**：`curl -s "https://www.81tiyu.com/TV/bf.aspx?id={id}"`
